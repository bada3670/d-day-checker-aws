## 사용한 서비스 및 동작 원리

다음 서비스들을 사용하였습니다.

EC2, RDS

RDS에서는 MySQL을 운영하고 있고, EC2에서는 client와 server를 운영하고 있습니다.

client는 webpack의 dev-server를 사용하고 있고, server는 express를 사용하고 있습니다.

EC2에서 터미널을 꺼도 운영이 될 수 있게 리눅스의 screen을 사용하고 있습니다.
